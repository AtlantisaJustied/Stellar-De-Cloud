在我們的服務器，你可以和你的好友一起聯機！
### 不只是MC，原神也可以。
你只需要輸入一行神秘代碼 

> `/coop [userUID]`

輸入這條命令，可以使你強制進入好友的世界。

userUID就是你的好友的UID，它會在右下角顯示，跟原版一樣。
